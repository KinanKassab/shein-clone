# Category Icons

Please add high-quality PNG icon files for each category:

- `men.png` - Icon for Men's category
- `women.png` - Icon for Women's category
- `children.png` - Icon for Children's category
- `baby.png` - Icon for Baby's category
- `electronics.png` - Icon for Electronics category
- `cars.png` - Icon for Cars category
- `home.png` - Icon for Home category

**Recommended specifications:**
- Format: PNG with transparency
- Size: 64x64px to 128x128px
- Style: Professional, modern, and consistent across all icons
- Background: Transparent or solid color matching the design

The icons will automatically fallback to emoji if the PNG files are not found.
